listas ordenadas

se usa a tag ol para não precisar ordenar manualmente, usar o parametro dentro dessa tag chamado type te permite escolher o tipo de numeração ou ordemanento da lista alem de escolher por onde começar usando o parametro start(4 para nao começar do 1 ou i) e usando a tag li para informar que é uma item da lista

  1. algo
  2. isso
  3. aquilo

listas não ordenadas

listas não ordenadas usam a tag ul para marcar um ponto nos tópicos da lista e podem receber o parametro type sendo disc para bolinha preta, circle para um circulo vazado e square para um quadrado e posso fazer com que os itens de outra bulistas continuem a contagem da primeira usando o parametro start

  • algo
  • isso
  • aquilo
